## **Master Prompt to Recreate the Automation Tribe Two-Step AI Interface**

**Prompt:**

````
I want you to create a complete, production-ready HTML file with inline JavaScript and TailwindCSS that implements my **Automation Tribe Two-Step AI Workflow Interface**.  

### 1) Overview & Workflow
The application connects to a **single n8n webhook** via a constant named:

```js
const WEBHOOK_URL = "YOUR_N8N_WEBHOOK_URL_HERE";
````

**Important:**

* This one webhook handles **two separate tasks**.
* n8n uses a **Switch node** to route between them.
* The Switch checks the value of `body.action` (or `body.task`) in the incoming JSON/form-data.
* The two routes are:

**Route 1: "Generate Prompts"**

* Switch condition: `action` = `"generate_prompts"`.
* The n8n workflow uses AI to return an array of prompts.
* **Response format** from n8n:

```json
{
  "prompts": [
    "Prompt 1 full text here...",
    "Prompt 2 full text here...",
    "... up to 5 prompts ..."
  ]
}
```

**Route 2: "Generate Image"**

* Switch condition: `action` = `"generate_images"`.
* n8n takes a selected prompt + uploaded image and returns a **binary image file**.
* The **Respond to Webhook** node is set to:

  * Respond With: **Binary File**
  * Binary Property: `data` (contains the generated image)
  * MIME Type: `image/png` (or `image/jpeg`)
  * File Name: `generated.png`

#### Step 1: Idea & Prompt Generation

* Large textarea with placeholder: *"Enter a simple idea..."*.
* Live character counter under the textarea.
* Buttons:

  1. **Generate Ideas**:

     * Enabled only when textarea has content.
     * Sends JSON POST to `WEBHOOK_URL`:

       ```json

         "initial_prompt": "<textarea value>"
       }
       ```
     * On success, expects `prompts[]` array (5 items).
     * Moves to Step 2 with these prompts displayed.
  2. **Use This Prompt**:

     * Sends user directly to Step 2 using textarea content as the selected prompt.
     * Skips prompt generation.

  * Remove button to clear file.
* **Generate Image Button**:

  * Enabled only if a prompt is selected (or prefilled) **and** an image is uploaded.
  * Sends `multipart/form-data` to `WEBHOOK_URL` with:

    * `action`: `"generate_images"`
    * `task`: `"generate_images"`
    * `final_prompt`: selected prompt text
    * `size`: selected size enum
    * `data`: binary image file (resized if needed before send)
